**рЃАрЃЎрЃљрЃБрЃбрЃўрЃюрЃњрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃА (Scouting System)**, рЃАрЃљрЃЊрЃљрЃф рЃЌрЃљрЃЋрЃЏрЃЮрЃДрЃарЃўрЃџрЃўрЃљ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ рЃцрЃћрЃ«рЃЉрЃБрЃарЃЌрЃћрЃџрЃћрЃЉрЃќрЃћ, рЃЏрЃљрЃЌ рЃЎрЃџрЃБрЃЉрЃћрЃЉрЃќрЃћ, рЃљрЃњрЃћрЃюрЃбрЃћрЃЉрЃАрЃљ рЃЊрЃљ рЃАрЃбрЃљрЃбрЃўрЃАрЃбрЃўрЃЎрЃљрЃќрЃћ.

---

### ­ЪЊЮ рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ: "Football Scouting DB"

4 рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃЎрЃџрЃљрЃАрЃў рЃЊрЃљ рЃЊрЃљрЃљрЃЎрЃљрЃЋрЃерЃўрЃарЃЮ рЃўрЃАрЃўрЃюрЃў Code First рЃЏрЃўрЃЊрЃњрЃЮрЃЏрЃўрЃЌ.

#### 1. рЃАрЃљрЃГрЃўрЃарЃЮ рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў (Entities):

рЃерЃћрЃЦрЃЏрЃћрЃюрЃў рЃћрЃА рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў рЃЊрЃљ рЃњрЃљрЃБрЃгрЃћрЃарЃћ `[Key]`, `[Required]` рЃЊрЃљ `[MaxLength]` рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃћрЃЉрЃў рЃАрЃљрЃЊрЃљрЃф рЃАрЃљрЃГрЃўрЃарЃЮрЃљ:

1. **Club (рЃАрЃљрЃцрЃћрЃ«рЃЉрЃБрЃарЃЌрЃЮ рЃЎрЃџрЃБрЃЉрЃў)**

- `Id`
- `Name` (рЃЏрЃљрЃњ: "Napoli", "Dinamo Tbilisi")
- `Country` (рЃЦрЃЋрЃћрЃДрЃљрЃюрЃљ)

2. **Footballer (рЃцрЃћрЃ«рЃЉрЃБрЃарЃЌрЃћрЃџрЃў)**

- `Id`
- `FirstName`
- `LastName`
- `JerseyNumber` (рЃЏрЃљрЃўрЃАрЃБрЃарЃўрЃА рЃюрЃЮрЃЏрЃћрЃарЃў)
- `BirthDate` (рЃЊрЃљрЃЉрЃљрЃЊрЃћрЃЉрЃўрЃА рЃЌрЃљрЃарЃўрЃдрЃў)

3. **Agent (рЃљрЃњрЃћрЃюрЃбрЃў)**

- `Id`
- `Name` (рЃЏрЃљрЃњ: "Mamuqa Jugeli")
- `AgencyName` (рЃАрЃљрЃљрЃњрЃћрЃюрЃбрЃЮрЃА рЃАрЃљрЃ«рЃћрЃџрЃў)

4. **TransferMarketData (рЃАрЃљрЃбрЃарЃљрЃюрЃАрЃцрЃћрЃарЃЮ рЃЏрЃљрЃарЃЎрЃћрЃбрЃўрЃА рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў)**

- `Id`
- `MarketValue` (рЃдрЃўрЃарЃћрЃЉрЃБрЃџрЃћрЃЉрЃљ, рЃЏрЃљрЃњ: 80000000)
- `ContractExpirationDate` (рЃЎрЃЮрЃюрЃбрЃарЃљрЃЦрЃбрЃўрЃА рЃЊрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃА рЃЌрЃљрЃарЃўрЃдрЃў)

---

#### 2. рЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃў (Relationships), рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃБрЃюрЃЊрЃљ рЃљрЃљрЃгрЃДрЃЮ:

рЃДрЃБрЃарЃљрЃЊрЃдрЃћрЃЉрЃўрЃЌ рЃЏрЃўрЃДрЃћрЃЋрЃў рЃџрЃЮрЃњрЃўрЃЎрЃљрЃА:

**1. One-to-Many (1xM): рЃЎрЃџрЃБрЃЉрЃў рЃЊрЃљ рЃцрЃћрЃ«рЃЉрЃБрЃарЃЌрЃћрЃџрЃћрЃЉрЃў**

- рЃћрЃарЃЌ **рЃЎрЃџрЃБрЃЉрЃА** (рЃЏрЃљрЃњ: Napoli) рЃ░рЃДрЃљрЃЋрЃА рЃЉрЃћрЃЋрЃарЃў **рЃцрЃћрЃ«рЃЉрЃБрЃарЃЌрЃћрЃџрЃў** (Kvara, Osimhen...).
- рЃћрЃарЃЌ **рЃцрЃћрЃ«рЃЉрЃБрЃарЃЌрЃћрЃџрЃА** рЃљрЃЦрЃЋрЃА рЃЎрЃЮрЃюрЃбрЃарЃљрЃЦрЃбрЃў рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃћрЃарЃЌ **рЃЎрЃџрЃБрЃЉрЃЌрЃљрЃю**.
- _рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ:_ `Footballer` рЃЎрЃџрЃљрЃАрЃерЃў рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћ `ClubId`.

**2. One-to-One (1x1): рЃцрЃћрЃ«рЃЉрЃБрЃарЃЌрЃћрЃџрЃў рЃЊрЃљ рЃАрЃљрЃбрЃарЃљрЃюрЃАрЃцрЃћрЃарЃЮ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃў**

- рЃДрЃЋрЃћрЃџрЃљ **рЃцрЃћрЃ«рЃЉрЃБрЃарЃЌрЃћрЃџрЃА** рЃљрЃЦрЃЋрЃА рЃќрЃБрЃАрЃбрЃљрЃЊ рЃћрЃарЃЌрЃў рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃў **TransferMarketData**-рЃерЃў (рЃАрЃљрЃЊрЃљрЃф рЃгрЃћрЃарЃўрЃљ рЃЏрЃўрЃАрЃў рЃцрЃљрЃАрЃў рЃЊрЃљ рЃЎрЃЮрЃюрЃбрЃарЃљрЃЦрЃбрЃўрЃА рЃЋрЃљрЃЊрЃљ).
- рЃћрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃцрЃћрЃ«рЃЉрЃБрЃарЃЌрЃћрЃџрЃўрЃА рЃфрЃ«рЃарЃўрЃџрЃерЃў рЃарЃЮрЃЏ рЃљрЃа рЃћрЃгрЃћрЃарЃЮрЃА (рЃарЃљрЃЊрЃњрЃљрЃю рЃ«рЃерЃўрЃарЃљрЃЊ рЃўрЃфрЃЋрЃџрЃћрЃЉрЃљ рЃЊрЃљ рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃфрЃљрЃџрЃЎрЃћ рЃАрЃћрЃарЃЋрЃўрЃАрЃўрЃЊрЃљрЃю рЃЏрЃЮрЃЊрЃўрЃЮрЃЊрЃћрЃА), рЃфрЃљрЃџрЃЎрЃћ рЃфрЃ«рЃарЃўрЃџрЃљрЃЊ рЃњрЃљрЃЏрЃўрЃбрЃљрЃюрЃћрЃЌ.
- _рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ:_ `TransferMarketData` рЃўрЃДрЃЮрЃА рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃў рЃЎрЃџрЃљрЃАрЃў (Foreign Key рЃЏрЃљрЃАрЃерЃў рЃўрЃДрЃЮрЃА).

**3. Many-to-Many (MxM): рЃцрЃћрЃ«рЃЉрЃБрЃарЃЌрЃћрЃџрЃў рЃЊрЃљ рЃљрЃњрЃћрЃюрЃбрЃў**

- _рЃљрЃЦ рЃфрЃЮрЃбрЃљ рЃЕрЃљрЃ«рЃџрЃљрЃарЃЌрЃБрЃџрЃў рЃАрЃўрЃбрЃБрЃљрЃфрЃўрЃљрЃљ:_ рЃарЃћрЃљрЃџрЃБрЃа рЃфрЃ«рЃЮрЃЋрЃарЃћрЃЉрЃљрЃерЃў рЃцрЃћрЃ«рЃЉрЃБрЃарЃЌрЃћрЃџрЃА 1 рЃљрЃњрЃћрЃюрЃбрЃў рЃ░рЃДрЃљрЃЋрЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃљрЃЏ рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃгрЃљрЃарЃЏрЃЮрЃЋрЃўрЃЊрЃњрЃўрЃюрЃЮрЃЌ, рЃарЃЮрЃЏ рЃцрЃћрЃ«рЃЉрЃБрЃарЃЌрЃћрЃџрЃА рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃ░рЃДрЃљрЃЋрЃЊрЃћрЃА рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃгрЃљрЃарЃЏрЃЮрЃЏрЃљрЃЊрЃњрЃћрЃюрЃћрЃџрЃў (рЃЏрЃљрЃњ: рЃћрЃарЃЌрЃў рЃўрЃБрЃарЃўрЃАрЃбрЃў, рЃћрЃарЃЌрЃў рЃАрЃърЃЮрЃарЃбрЃБрЃџрЃў рЃљрЃњрЃћрЃюрЃбрЃў) рЃЊрЃљ рЃћрЃарЃЌ рЃљрЃњрЃћрЃюрЃбрЃА рЃ░рЃДрЃљрЃЋрЃА рЃЉрЃћрЃЋрЃарЃў рЃцрЃћрЃ«рЃЉрЃБрЃарЃЌрЃћрЃџрЃў.
- _рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљ:_ Fluent API-рЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ, рЃерЃБрЃљрЃЏрЃљрЃЋрЃљрЃџ рЃфрЃ«рЃарЃўрЃџрЃА рЃЉрЃљрЃќрЃљрЃерЃў рЃЊрЃљрЃљрЃарЃЦрЃЋрЃў **`ClientRelations`** (рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў `AgentFootballer`-рЃўрЃА рЃюрЃљрЃфрЃЋрЃџрЃљрЃЊ).

---

#### 3. рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃў рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃљ:

1. **рЃерЃћрЃЦрЃЏрЃћрЃюрЃў рЃљрЃ«рЃљрЃџрЃў рЃърЃарЃЮрЃћрЃЦрЃбрЃў** рЃЊрЃљ рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћ EF Core рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃў.
2. **рЃЊрЃљрЃгрЃћрЃарЃћ рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў.**

- рЃЎрЃџрЃБрЃЉрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў (`Name`) рЃњрЃљрЃ«рЃљрЃЊрЃћ рЃАрЃљрЃЋрЃљрЃџрЃЊрЃћрЃЉрЃБрЃџрЃЮ `[Required]`.

3. **рЃЊрЃљрЃљрЃЎрЃЮрЃюрЃцрЃўрЃњрЃБрЃарЃўрЃарЃћ DbContext.**

- 1xM рЃЎрЃљрЃЋрЃерЃўрЃарЃў рЃњрЃљрЃљрЃЎрЃћрЃЌрЃћ рЃўрЃАрЃћ, рЃарЃЮрЃЏ рЃЌрЃБ рЃЎрЃџрЃБрЃЉрЃў рЃгрЃљрЃўрЃерЃџрЃћрЃЉрЃљ, рЃцрЃћрЃ«рЃЉрЃБрЃарЃЌрЃћрЃџрЃћрЃЉрЃў рЃљрЃа рЃгрЃљрЃўрЃерЃљрЃџрЃЮрЃА (рЃљрЃюрЃБ `OnDelete`-рЃќрЃћ рЃЏрЃўрЃБрЃЌрЃўрЃЌрЃћ `SetNull` рЃљрЃю `Restrict`, рЃЌрЃБ рЃўрЃфрЃў рЃарЃЮрЃњрЃЮрЃа. рЃЌрЃБ рЃљрЃарЃљ рЃЊрЃљ, рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў рЃЊрЃљрЃбрЃЮрЃЋрЃћ).
- MxM рЃЎрЃљрЃЋрЃерЃўрЃарЃўрЃА рЃфрЃ«рЃарЃўрЃџрЃўрЃА рЃАрЃљрЃ«рЃћрЃџрЃў рЃерЃћрЃфрЃЋрЃљрЃџрЃћ `OnModelCreating`-рЃерЃў.

4. **Migration:**

- рЃерЃћрЃЦрЃЏрЃћрЃюрЃў рЃЏрЃўрЃњрЃарЃљрЃфрЃўрЃљ рЃАрЃљрЃ«рЃћрЃџрЃўрЃЌ: `InitialFootballDB`.
- рЃњрЃљрЃюрЃљрЃљрЃ«рЃџрЃћ рЃЉрЃљрЃќрЃљ.

#### 4. (рЃЉрЃЮрЃюрЃБрЃАрЃў) рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃЕрЃљрЃгрЃћрЃарЃљ:

`Program.cs`-рЃерЃў рЃЊрЃљрЃгрЃћрЃарЃћ рЃЎрЃЮрЃЊрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЉрЃљрЃќрЃљрЃерЃў рЃЕрЃљрЃгрЃћрЃарЃА:

1. рЃЎрЃџрЃБрЃЉрЃА: **"Napoli"**.
2. рЃљрЃњрЃћрЃюрЃбрЃА: **"Jugeli"**.
3. рЃцрЃћрЃ«рЃЉрЃБрЃарЃЌрЃћрЃџрЃА: **"Khvicha Kvaratskhelia"**, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЌрЃљрЃЏрЃљрЃерЃЮрЃЉрЃА **Napoli**-рЃерЃў, рЃ░рЃДрЃљрЃЋрЃА рЃљрЃњрЃћрЃюрЃбрЃў **Jugeli** рЃЊрЃљ рЃЏрЃўрЃАрЃў рЃАрЃљрЃбрЃарЃљрЃюрЃАрЃцрЃћрЃарЃЮ рЃцрЃљрЃАрЃўрЃљ **80,000,000**.

рЃљрЃЉрЃљ, рЃњрЃћрЃџрЃЮрЃЊрЃћрЃЉрЃў! рЃЎрЃЮрЃЊрЃў рЃЊрЃљрЃгрЃћрЃарЃћ рЃЊрЃљ рЃЌрЃБ рЃАрЃљрЃЊрЃЏрЃћ рЃњрЃљрЃўрЃГрЃћрЃЊрЃћрЃЉрЃў (рЃњрЃљрЃюрЃАрЃљрЃЎрЃБрЃЌрЃарЃћрЃЉрЃўрЃЌ рЃЉрЃЮрЃюрЃБрЃА рЃюрЃљрЃгрЃўрЃџрЃерЃў, рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃерЃћрЃюрЃљрЃ«рЃЋрЃўрЃАрЃљрЃА), рЃљрЃЦ рЃЋрЃљрЃа.
